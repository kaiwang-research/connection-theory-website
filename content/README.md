# Content Directory

This directory contains all documentation used by the Connection Theory website.

## Structure

```text
content/
├── concepts/     Official concept definitions
├── papers/       Paper pages and publication materials
├── archive/      Historical drafts and internal documents

Directory Guidelines
concepts/

Contains the canonical definitions of Connection Theory's foundational concepts.

These documents serve as the single source of truth for terminology used throughout the website and research papers.

papers/

Contains documentation for individual papers, including summaries, publication information, figures, and supporting materials.

archive/

Contains historical drafts, retired materials, and internal working documents.

Files in this directory are preserved for reference but are not intended to be published on the website by default.


---

## Maintenance Principles

- Canonical concept definitions belong in `concepts/`.
- Individual papers should reference concepts rather than redefine them.
- Historical drafts should be moved to `archive/` instead of remaining alongside official documents.
